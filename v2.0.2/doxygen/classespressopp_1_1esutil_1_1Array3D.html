Detailed Description

template<class T, OutlierMode outlier_mode = exception>
class espressopp::esutil::Array3D< T, outlier_mode >

A three-dimensional array (i.e.

a matrix). Prticulary we need it for 3-body potential tables

Parameters
TThe type of the object stored in the array.
Example:
// create the 10x10 id matrix
Array3D< double > I(10, 10, ,10, 0.0);
for (int i = 0; i < 10; i++) I(i, i, i) = 1.0;
