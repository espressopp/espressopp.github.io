bool espressopp::FixedQuadrupleAngleList::add ( longint  pid1,
longint  pid2,
longint  pid3,
longint  pid4 
)

Add the given particle quadruple to the list on this processor if the particle with the lower id belongs to this processor.

Note that this routine does not check whether the quadruple is inserted on another processor as well.

it contains the information about the initial angle for each quadruple.

This is probably temporary fast solution for the dihedral potential where all the quadruples have different phi0

Returns:
whether the quadruple was inserted on this processor.
