26  /*
27  * LiquidGasLB constructor expects 5 parameters (and a system pointer).
28  * These are: lattice size in 3D Ni, lattice spacing a, lattice timestep tau,
29  * number of dimensions and number of velocity vectors on a lattice site.
30  * The lattice size, Ni, is an obligatory parameter and must be set at the
31  * beginning of the simulation.
32  *
33  * The default lattice model is D3Q19 and both lattice spacing and timestep
34  * are set to 1.
35  *
36  * Note that at the present stage of development we aim at D3Q19 model.
37  * If you want to use something else, please, feel free to modify the code.
38  *
39  * Originally, we had planned this module to operate in 3D only, so if you
40  * need a 2D version, there is a bit more tuning involved. On the other hand,
41  * adding different 3D lattice models (such as D3Q15 or D3Q27) is rather
42  * straightforward.
43  *
44  */
