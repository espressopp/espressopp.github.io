42  /* LatticeBoltzmann constructor expects 5 parameters (and a system pointer).
43  These are: lattice size in 3D Ni, lattice spacing a, lattice timestep tau,
44  number of dimensions and number of velocity vectors on a lattice site.
45  The lattice size, Ni, is an obligatory parameter and must be set at the
46  beginning of the simulation.
47 
48  The default lattice model is D3Q19 and both lattice spacing and timestep
49  are set to 1.
50 
51  Note that at the present stage of development we aim at D3Q19 model.
52  If you want to use something else, please, feel free to modify the code.
53 
54  Originally, we had planned this module to operate in 3D only, so if you
55  need a 2D version, there is a bit more tuning involved. On the other hand,
56  adding different 3D lattice models (such as D3Q15 or D3Q27) is rather
57  straightforward.
58  */
