Returns the element at position pos.

pos must be a vector where the expressions pos[0], pos[1] and pos[2] return the first and second element. 

When the element doesn't exist, the Array3D is automatically
extended so that the element exists. The new elements are

Changes the size of the Array3D so that the size becomes n x m x l.

Attention
The elements of the Array3D are not initialised consistently, so the values are undefined. The t parameter exists only for the purpose to provide a default T value when new elements are created and needs to be specified only when T has no default constructor.
