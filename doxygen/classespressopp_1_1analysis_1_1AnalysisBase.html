Detailed Description

All quantities to be measured derive from this abstract base class and the corresponding template.

The following functions have to be implemented by the derived class: virtual ResultType computeRaw() virtual python::list compute() virtual python::list getAverageValue() virtual void resetAverage() virtual void updateAverage(ResultType res) = 0;
