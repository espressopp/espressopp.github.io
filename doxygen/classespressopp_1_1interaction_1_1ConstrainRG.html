Detailed Description

This class provides methods to compute forces and energies of the ConstrainRG part.

The code is based on G. Zhang's work. Reference in literature G. Zhang, L. A. Moreira, T. Stuehn, K. CH. Daoulas, K. Kremer, Macro Lett., 2014, 3, 198
