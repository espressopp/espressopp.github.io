Detailed Description

template<class T, OutlierMode outlier_mode = exception>
class espressopp::esutil::Array2D< T, outlier_mode >

A two-dimensional array (i.e.

a matrix).

Parameters
TThe type of the object stored in the array.
Example:
// create the 10x10 id matrix
Array2D< double > I(10, 10, 0.0);
for (int i = 0; i < 10; i++) I(i, i) = 1.0;

Changes the size of the Array2D so that the size becomes n x m.

Attention
The elements of the Array2D are not initialised consistently, so the values are undefined. The t parameter exists only for the purpose to provide a default T value when new elements are created and needs to be specified only when T has no default constructor.
